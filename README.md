### Run Locally

To run **HiBits** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/HiBits.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/HiBits.git
```
